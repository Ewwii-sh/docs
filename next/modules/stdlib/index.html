Std Library Module

These are all the standard modules in ewwii.

Each library in this module is under std::<m>, where <m> is the name of the specific module. These modules provide essential functionalities that are will be useful for making widgets. They cover tasks like string manipulation, environmental variable manipuation, running shell commands, and more.

Module

env

Namespace: global/std/env

fn get_current_dir​

fn get_current_dir() -> String
  • Description
  • Returns
  • Example

Get the current working directory.

This function returns the current working directory as a String. If there is an error (e.g., if the path cannot be retrieved), it returns a Result::Err with the error message.

import "std::env" as env;

// Get the current working directory
let current_dir = env::get_current_dir();
print(current_dir); // output: /home/username/project

fn get_env​

fn get_env(var: String) -> String
  • Description
  • Arguments
  • Returns
  • Example

Get the value of an environment variable.

  • var - The name of the environment variable to retrieve.

This function returns the value of the environment variable as a String. If the variable is not found or there is an error, it returns a Result::Err with the error message.

import "std::env" as env;

// Get the value of the "HOME" environment variable
let home_dir = env::get_env("HOME");
print(home_dir); // output: /home/username

  • Arguments
  • Returns
  • Example

regex

Namespace: global/std/regex

A module providing regular expression support.

fn find​

fn find(text: String, pattern: String) -> String
  • Description
  • Arguments
  • Returns
  • Example

Returns the first match of a regex pattern in the text.

  • text - A string to be matched with the pattern.
  • pattern - The pattern to match on the string.

Returns a string which is the first match of a regex pattern.

import "std::regex" as regex;

let result = regex::find("This is an example!", `\be\w*\b`);
print(result); // output: "example"

fn find_all​

fn find_all(text: String, pattern: String) -> Array
  • Description
  • Arguments
  • Returns
  • Example

Returns all matches of a regex pattern in the text.

  • text - A string to be matched with the pattern.
  • pattern - The pattern to match on the string.

Returns aan array of strings containing the all the things that match the regex pattern in the provided text.

import "std::regex" as regex;

let result = regex::find("This is an example!", `\be\w*\b`);
print(result); // output: ["example"]

fn is_match​

fn is_match(text: String, pattern: String) -> bool
  • Description
  • Arguments
  • Returns
  • Example

Checks if a regex pattern matches the given text.

  • text - A string to be matched with the pattern.
  • pattern - The pattern to match on the string.

Returns a boolean (true/false) based on if the pattern is matched on the text provided.

import "std::regex" as regex;

let result = regex::is_match("This is an example!", "example");
if result == true {
    print("The pattern is matched!");
}

fn replace​

fn replace(text: String, pattern: String, replacement: String) -> String
  • Description
  • Arguments
  • Returns
  • Example

Replaces all matches of a regex pattern with a replacement string.

  • text - A string to be matched with the pattern.
  • pattern - The pattern to match on the string.
  • replacement - A string that the matched pattern will get replaced with.

Returns the provided text with the matches of the regex pattern replaced with the provided replacement argument.

import "std::regex" as regex;

let result = regex::replace("This is an example!", "an example", "a test");
print(result); // output: "This is a test!"
Module

command

Namespace: global/std/command

fn run​

fn run(cmd: String)
  • Description
  • Arguments
  • Returns
  • Example

Executes a shell command without capturing the output.

  • cmd - The shell command to execute as a string.

This function returns nothing if the command executes successfully. If there is an error running the command, it returns the error.

import "std::command" as cmd;

// Run a shell command (e.g., list directory contents)
cmd::run("ls -l");

fn run_and_read​

fn run_and_read(cmd: String) -> String
  • Description
  • Arguments
  • Returns
  • Example

Executes a shell command and captures its output.

  • cmd - The shell command to execute as a string.

This function returns the standard output of the command as a string. If the command fails, it returns the error.

import "std::command" as cmd;

// Run a shell command and capture its output
let output = cmd::run_and_read("echo 'Hello, world!'");
print(output); // output: Hello, world!
